Руководство по добавлению различных поведенческих конфигураций для Джарвиса
    Джарвис имеет несколько стилей общения, и вы можете самолично добавить новый.
Это делается добавлением нового файла конфигигурации роли, которые находятся в папке
/app/modules/core/roles/ (там же, где и эта инструкция).
    Файл конфига для определенной личности Джарвиса содержит:
     - Имя
     - Системное имя
     - ID (кастомный конфиг не может содержать ID, равное 0!)
     - Слова активации
     - Промпт
     - Приветственное сообщение
     - Вариации ответов на:
        - приветствие
        - выполнение команды
        - похвалу
        - оскорбления
        - отсутствие запроса
        - команду выключения
     - Вариант голоса(озвучки), из следующего списка:
        - aidar
        - baya
        - kseniya
        - xenia
        - random
    Пример того, как должен выглядеть стилевой конфиг, можно увидеть в файле /app/modules/core/roles/example_config.py,
или посмотрев на любой уже существующий стилевой конфиг.
    Рекомендации по наименованию конфига: называйте файл конфига в формате [имя персонажа]_config.py, т.к. программа
считывает именно первую букву названия файла в качестве сокращения названия конфига в меню выбора роли.
